Detailed Description

HTTP packet creator.

This Class is responsible for handling the message buffer containing a HTTP request. After parsing the request the user can ask the class to process the message with the file selected under the HTTP Header for GET. A JSON message telling if the file was successfully added or removed for POST, PUT and DELETE.
