Funcionamento

Ao ser executado, o servidor aguarda uma nova conexão.

Ao receber tal conexão via TCP, o servidor iniciará uma thread que irá tratar a solicitação HTTP do cliente.

Esta thread se comunica com a classe HTTP que irá gerar o pacote com base na solicitação, em seguida, o pacote gerado é enviado para a classe TCP que envia a informação ao cliente.

Características do Projeto

Método GET

Função GET funcionando normalmente. Ótimo para sites do tipo OnePage Layout

Verbose

Pode-se escolher se deseja ver um log durante o funcionamento ou não.

Método POST

Inserção de dados num simples banco de dados.

Método PUT

Edição de dados já existentes.

Método DELETE

Deleção de dados salvos.

Tratamento de Erros

Exceções customizadas em caso de ocorrência de erros.

Carregamento de Vídeo

Metódo POST
